# Advanced Calculator

A feature-rich, responsive web calculator built with HTML, CSS, and JavaScript. Uses Math.js for robust expression parsing and evaluation.

## Features
- Basic arithmetic (+, −, ×, ÷)
- Advanced functions: sin, cos, tan, ln, log10, sqrt, exponentiation (^)
- Memory functions: MC, MR, M+, M-
- ANS (last answer) support
- Keyboard input and shortcuts (Enter to evaluate, Backspace to delete)
- Small history of recent calculations
- Light / Dark theme toggle
- Responsive layout for mobile and desktop

## How to use
1. Open `index.html` in your browser.
2. Click buttons or use your keyboard to enter expressions.
3. Use `=` or Enter to evaluate.

## Files
- `index.html` — main page
- `style.css` — styling
